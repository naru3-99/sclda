# sclda
System Call Level Dynamic Analysis

## 取得する情報について
- システムコールに関連する情報
  - システムコールのID
  - システムコールの引数・返り値に関連する情報
  - 呼び出されたタイミングについての情報
    - プロセスがユーザ空間で消費したCPU時間
    - プロセスがカーネル空間で消費したCPU時間
  - プロセスのメモリ消費量に関連する情報
    - スタック
    - ヒープ
    - メモリ全体
- プロセス生成に関連する情報
  - 生成されたプロセスのID(PID)
  - 生成されたプロセスの親プロセスのID(PPID)
